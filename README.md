# Java Foundation Project - DataProcessor

A Java project built to learn core Java concepts used in 
enterprise automation frameworks.

## Tech Stack
- Java 17
- Maven

## Project Structure
src/
└── main/
    └── java/
        └── com/dataprocessor/
            ├── model/       # Data models
            ├── processor/   # Business logic
            ├── reader/      # Data reading
            └── util/        # Utility classes

## Classes Built
- `Student.java` - Model class with encapsulation, 
   constructor, getters, setters, toString

## Concepts Covered
- OOP - Classes and Objects
- Encapsulation
- Constructors
- Maven project structure
- Package organisation
